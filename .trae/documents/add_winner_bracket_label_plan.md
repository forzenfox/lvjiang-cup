# 淘汰赛添加"胜者组"标签计划

## 问题描述
当前淘汰赛页面左侧有"败者组"标签，但右侧胜者组区域缺少对应的"胜者组"标签，视觉上不够统一。

## 目标
在胜者组区域添加"胜者组"标签，与"败者组"标签保持视觉风格一致。

## 实现方案

### 1. 分析当前布局
- 败者组标签位置：`left-[20px] top-[300px]`
- 胜者组比赛卡片位置：
  - g1: { x: 20, y: 20 }
  - g2: { x: 20, y: 160 }
  - g5: { x: 20 + xSpacing(280), y: 90 }
  - g8: { x: 20 + xSpacing*3, y: 250 }

### 2. 胜者组标签位置设计
考虑到胜者组包含 g1、g2（第一轮）和 g5（第二轮胜者组决赛）、g8（总决赛），建议将"胜者组"标签放置在：
- 位置：`left-[20px] top-[100px]`（在 g1 和 g2 之间偏上的位置）
- 样式：与"败者组"标签保持一致

### 3. 具体修改
文件：`src/components/features/EliminationStage.tsx`

在败者组标签之前添加胜者组标签：
```tsx
<div className="absolute left-[20px] top-[100px] text-gray-500 font-bold text-sm tracking-widest border-l-4 border-gray-700 pl-2">
  胜者组
</div>
```

## 验证步骤
1. 启动开发服务器
2. 切换到淘汰赛标签页
3. 确认"胜者组"标签显示在左上方区域
4. 确认"败者组"标签仍然显示在左下方区域
5. 确认两个标签样式一致
