# 使用 DevTools 创建战队信息计划

## 目标
使用 Chrome DevTools MCP 工具访问 http://localhost:5173/admin/teams，按照模拟数据创建战队信息。

## 前置条件
- 开发服务器运行在 http://localhost:5173
- 管理员账号已配置（密码：admin123）

## 执行步骤

### 步骤 1: 访问战队管理页面
1. 使用 `mcp_Chrome_DevTools_MCP_navigate_page` 导航到 http://localhost:5173/admin/teams
2. 检查页面状态，确认是否需要登录

### 步骤 2: 登录（如需要）
1. 如果页面显示登录表单，使用管理员密码 `admin123` 登录
2. 使用 `mcp_Chrome_DevTools_MCP_fill` 填写密码
3. 使用 `mcp_Chrome_DevTools_MCP_click` 点击登录按钮

### 步骤 3: 获取模拟数据
1. 读取 `src/mock/data.ts` 文件中的初始战队数据
2. 提取战队信息（名称、队标、简介、队员等）

### 步骤 4: 循环创建战队
对于每个模拟数据中的战队：
1. 点击"添加战队"按钮
2. 填写战队名称
3. 填写队标链接（如有）
4. 填写战队简介（如有）
5. 填写5个固定位置的队员信息（上单、打野、中单、AD、辅助）
6. 点击"保存战队"按钮
7. 等待保存完成，验证战队是否创建成功

### 步骤 5: 验证结果
1. 截图确认所有战队已正确创建
2. 检查战队列表显示

## 模拟数据来源
- 文件：`src/mock/data.ts`
- 初始战队数据：`initialTeams` 数组

## 注意事项
- 每个战队需要填写5个固定位置的队员
- 保存后需要等待页面刷新确认创建成功
- 如遇错误需要记录并处理
