## 实施步骤

1. **修改数据结构**
   - 在 Match 类型中添加 `stage` 字段（'swiss' | 'elimination'）
   - 更新 mock 数据，区分瑞士轮和淘汰赛

2. **创建页签组件**
   - 使用 Tabs 组件创建"瑞士轮"和"淘汰赛"两个页签
   - 添加切换动画效果

3. **重构瑞士轮视图**
   - 按天数分组显示比赛
   - 显示战绩（0-0, 1-0, 0-1, 1-1, 0-2, 1-2, 2-0, 2-1）
   - 显示晋级/淘汰结果

4. **创建淘汰赛视图**
   - 双败赛制树状图
   - 胜者组在上方，败者组在下方
   - 显示总决赛

5. **更新数据模型**
   - 添加瑞士轮所有比赛数据
   - 添加淘汰赛所有比赛数据

## 数据结构修改

```typescript
// 在 Match 类型中添加
interface Match {
  // ... 现有字段
  stage: 'swiss' | 'elimination';
  swissRound?: number; // 瑞士轮轮次
  eliminationRound?: string; // 淘汰赛轮次
  bracket?: 'winners' | 'losers' | 'finals'; // 淘汰赛分组
}
```

## 组件结构

```
ScheduleSection
├── Tabs (瑞士轮 | 淘汰赛)
│   ├── SwissStage (瑞士轮视图)
│   │   ├── Day1Section
│   │   ├── Day2Section
│   │   ├── Day3Section
│   │   ├── Day4Section
│   │   └── AdvancementResults (晋级结果)
│   └── EliminationStage (淘汰赛视图)
│       ├── WinnersBracket (胜者组)
│       └── LosersBracket (败者组)
```

## Admin 管理界面修改

管理后台的赛程管理页面需要同步更新：

### 当前问题
- 只支持简单比赛列表
- 缺少阶段（瑞士轮/淘汰赛）筛选
- 缺少战绩和分组字段

### 修改内容
1. **添加阶段筛选器** - 下拉选择瑞士轮/淘汰赛
2. **添加阶段字段编辑** - 比赛属于哪个阶段
3. **添加战绩显示** - 瑞士轮显示当前战绩
4. **添加分组选择** - 淘汰赛区分胜者组/败者组
5. **优化列表展示** - 按阶段分组显示比赛

### 界面调整
```
AdminSchedule
├── 阶段筛选 Tabs (全部 | 瑞士轮 | 淘汰赛)
├── 瑞士轮管理
│   ├── 添加"战绩"列（0-0, 1-0等）
│   └── 按天数分组显示
└── 淘汰赛管理
    ├── 添加"分组"列（胜者组/败者组/总决赛）
    └── 按分组显示
```
