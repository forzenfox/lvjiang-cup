## 1. 文档概述

### 1.1 文档目的
本文档描述驴酱杯 LOL 娱乐赛事网站页脚功能的产品需求，用于指导设计和开发团队实现页脚优化功能。

### 1.2 产品背景
驴酱杯是一个仿照 LPL 官方赛事网站的 LOL 娱乐赛事网站，为斗鱼驴酱公会的主播和水友提供赛事信息展示平台。为了提升品牌曝光度和用户粘性，需要在网站底部添加社交媒体链接和联系方式。

### 1.3 需求背景
- 当前网站无页脚设计
- 需要为用户提供社交媒体关注渠道
- 需要展示合作联系方式
- 需要符合国内网站合规要求（ICP 备案）

---

## 2. 产品目标

### 2.1 核心目标
1. **品牌曝光**：通过社交媒体链接提升驴酱公会在各平台的影响力
2. **用户连接**：为用户提供多种关注渠道，便于获取最新赛事信息
3. **商业合作**：提供合作邮箱，便于商务对接
4. **合规要求**：展示 ICP 备案号，满足国内网站运营规范

### 2.2 成功指标
- 页脚在各页面正常展示（桌面端）
- 社交媒体链接可正常跳转
- 微信公众号二维码可正常显示
- 备案号正确显示（无需跳转）
- 移动端不显示页脚（符合产品需求）

## 4. UI/UX 设计需求

### 4.1 设计风格
页脚设计应与网站整体风格保持一致：

#### 4.1.1 色彩方案
- **背景色**：深蓝色 `#1E3A8A` 或深灰色 `#1F2937`
- **文字颜色**：白色 `#FFFFFF` 或浅灰色 `#D1D5DB`
- **链接颜色**：金色 `#F59E0B`（悬停时）/ 白色（默认）
- **图标颜色**：白色或各平台品牌色

#### 4.1.2 布局设计

**方案一：横向分布布局（推荐）**

充分利用 PC 端宽度优势，采用多列网格布局：

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  [B 站]        [抖音]        [抖音]         微信公众号： [驴驴电竞]       │
│  胡凯利_洞主    凯菇来啦      胡凯利_洞主     (悬停显示二维码)            │
│                                                                         │
│  ─────────────────────────────────────────────────────────────          │
│                                                                         │
│  合作邮箱：lvjiangshangwu@163.com          鄂 ICP 备 2026017374 号 -1      │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**布局说明**：
- **上半部分**：左侧 3 列社交媒体链接（均匀分布），右侧微信公众号（文字 + 悬停二维码）
- **分隔线**：使用金色细线分隔上下区域
- **下半部分**：左侧合作邮箱，右侧备案号（两端对齐）
- **整体宽度**：最大宽度限制在 1400px，居中显示
- **左右边距**：48px（充分利用宽度，同时保持呼吸感）

**微信公众号交互设计**：
```
默认状态：
  微信公众号：驴驴电竞

悬停状态：
  ┌─────────────────┐
  │  [二维码图片]    │ ← 浮层显示
  └─────────────────┘
  微信公众号：驴驴电竞
```

**方案二：居中对称布局**

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│              [B 站]  [抖音]  [抖音]                                      │
│            胡凯利_洞主  凯菇来啦  胡凯利_洞主                             │
│                                                                         │
│                     微信公众号：驴驴电竞                                 │
│                     [二维码图片]                                        │
│                                                                         │
│  ─────────────────────────────────────────────────────────────          │
│                                                                         │
│        合作邮箱：lvjiangshangwu@163.com          鄂 ICP 备 2026017374 号 -1│
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**布局说明**：
- 所有内容居中对齐
- 社交媒体链接横向排列在顶部
- 微信公众号居中展示
- 邮箱和备案号在下半部分左右分布

**推荐采用方案一**，理由：
- 更好地利用 PC 端宽度
- 信息层次更清晰
- 视觉重心更平衡
- 符合现代网页设计趋势

#### 4.1.3 间距规范
- **页脚内边距**：上下 48px，左右 48px（增加左右边距，充分利用宽度）
- **模块间距**：32px（增加间距，提升呼吸感）
- **图标间距**：24px（增加图标间距，避免拥挤）
- **图标尺寸**：32x32px（适当增大图标，提升视觉权重）
- **二维码尺寸**：120x120px（适配 PC 端展示）
- **分隔线高度**：1px，颜色：金色 `#F59E0B`，透明度 50%
- **文字大小**：
  - 平台名称：14px
  - 邮箱/备案号：13px，颜色：浅灰色 `#9CA3AF`

### 4.2 交互设计

#### 4.2.1 悬停效果
- **社交媒体链接**：颜色变为金色 `#F59E0B`，显示下划线
- **图标**：轻微放大（scale: 1.15）或亮度提升，添加 Y 轴向上位移 2px
- **微信公众号**：
  - 文字：显示下划线，颜色变为金色
  - 二维码：显示浮层图片（淡入动画）
- **邮箱和备案号**：无悬停效果（纯文本）

#### 4.2.2 动画效果
- 使用 Framer Motion 实现平滑过渡
- 过渡时间：0.2s - 0.3s
- 缓动函数：ease-in-out
- **微信公众号二维码**：淡入淡出动画（opacity: 0 → 1）

### 4.3 图标和图片资源
需要准备的资源：
1. **Bilibili 图标**（SVG 格式，32x32px）
   - 使用 CDN 地址：`https://cdn.baomitu.com/cdn-static/images/logos/bilibili.svg`
2. **抖音图标**（SVG 格式，32x32px，2 个相同图标用于不同账号）
   - 使用 CDN 地址：`https://cdn.baomitu.com/cdn-static/images/logos/douyin.svg`
3. **微信公众号二维码图片**（WebP 格式）
   - 文件路径：`/assets/lvlvdianjing.webp`
   - 已存在于项目中：`frontend/public/assets/lvlvdianjing.webp`
   - 显示方式：鼠标悬停时显示
4. **资源使用说明**：
   - 社交媒体图标：直接引用外部 CDN（baomitu.com）
   - 微信公众号二维码：使用项目本地资源（public/assets 目录）

## 5. 技术实现方案

### 5.1 组件结构

```
src/
├── components/
│   └── layout/
│       ├── Footer.tsx              # 页脚主组件
│       └── Footer/
│           ├── SocialLinks.tsx     # 社交媒体链接组件（3 列布局）
│           ├── WeChatSection.tsx   # 微信公众号组件（文字 + 悬停二维码）
│           ├── ContactInfo.tsx     # 联系信息组件（邮箱）
│           └── ICPInfo.tsx         # ICP 备案信息组件
│
├── config/
│   └── footer.ts                   # 页脚配置文件
│
└── public/
    └── assets/
        └── lvlvdianjing.webp       # 微信公众号二维码（本地资源）
```

### 5.2 技术选型
- **框架**：React + TypeScript（与项目技术栈一致）
- **样式**：Tailwind CSS
- **图标**：Lucide React + 外部 CDN SVG（B 站、抖音）
- **动画**：Framer Motion
- **图片加载**：
  - 社交媒体图标：外部 CDN（baomitu.com）
  - 微信公众号二维码：本地静态资源（`/assets/lvlvdianjing.webp`）

### 5.3 代码示例

#### 5.3.1 Footer 组件结构
```typescript
interface FooterProps {
  className?: string;
}

export const Footer: React.FC<FooterProps> = ({ className }) => {
  return (
    <footer className={className}>
      <SocialLinks />
      <ContactInfo />
      <WeChatQR />
      <ICPBadge />
    </footer>

};
```

```typescript

<footer className="hidden lg:block">
  {/* 页脚内容 */}
</footer>
```

## 6. 图片加载方案

### 6.1 技术架构

本项目页脚采用**混合图片加载方案**：

```
┌─────────────────────────────────────────────────────────┐
│                    用户浏览器                            │
└───────────────────┬─────────────────────────────────────┘
                    │
        ┌───────────┴───────────┐
        │                       │
        ▼                       ▼
┌──────────────┐      ┌─────────────────┐
│  baomitu CDN │      │  本地静态资源    │
│  (外部图标)   │      │  (public 目录)   │
│              │      │                 │
│ - B 站图标    │      │ - 微信公众号二维码│
│ - 抖音图标    │      │   (/assets/     │
└──────────────┘      │    lvlvdianjing. │
                      │    webp)         │
                      └─────────────────┘
```

### 6.2 CDN 配置说明

#### 6.2.1 本地静态资源（微信公众号二维码）

**资源路径**：
```
/assets/lvlvdianjing.webp
```

**文件位置**：
- 开发环境：`frontend/public/assets/lvlvdianjing.webp`
- 生产环境：打包后自动复制到 `dist/assets/lvlvdianjing.webp`

**加载方式**：
- Vite 自动处理 `public` 目录下的静态资源
- 使用相对路径引用，无需 CDN 配置

#### 6.2.2 baomitu CDN（外部图标）

**CDN 服务商**：BootCDN（baomitu.com）

**资源地址**：
- B 站图标：`https://cdn.baomitu.com/cdn-static/images/logos/bilibili.svg`
- 抖音图标：`https://cdn.baomitu.com/cdn-static/images/logos/douyin.svg`

**特点**：
- 国内稳定的公共 CDN 服务
- 无需项目维护，直接使用
- 适合常用第三方图标

### 6.3 优势与注意事项

#### 6.3.1 优势

- ✅ **简单高效**：微信公众号二维码使用本地资源，无需额外配置
- ✅ **CDN 加速**：外部图标使用 baomitu CDN，加载快速
- ✅ **版本控制**：本地资源随项目版本管理
- ✅ **离线可用**：本地资源不依赖外部服务

#### 6.3.2 注意事项

- ⚠️ **文件大小**：建议图片文件不超过 500KB（已优化的 WebP 格式）
- ⚠️ **外部依赖**：baomitu CDN 为外部服务，需确保稳定性
- ⚠️ **悬停体验**：二维码浮层需要良好的定位和动画效果
