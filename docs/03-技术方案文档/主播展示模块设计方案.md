# 驴酱杯主播展示模块设计方案

## 1. 设计背景

为了提升驴酱杯赛事的影响力和观众参与度，需要在主页面添加一个专门的模块，用于展示参赛的明星主播和邀请的嘉宾主播。该模块将展示主播的海报、个人简介以及直播间跳转入口，方便观众了解主播信息并直接进入直播间观看。

## 2. 设计目标

- **提升用户体验**：为观众提供直观、美观的主播信息展示
- **增强互动性**：提供直播间直接跳转功能，方便观众快速进入直播
- **保持一致性**：与现有页面风格保持统一，确保整体视觉协调
- **响应式设计**：适配不同设备屏幕尺寸
- **可配置性**：通过管理后台动态配置主播信息
- **滚动浏览**：支持鼠标左右滚动查看主播海报

## 3. 数据结构设计

### 3.1 主播类型枚举

```typescript
// 主播类型枚举
export enum StreamerType {
  INTERNAL = 'internal',  // 驴酱内部主播
  GUEST = 'guest',         // 嘉宾主播
}
```

### 3.2 主播数据模型

```typescript
// 主播数据模型
export interface Streamer {
  id: string;
  nickname: string;       // 主播昵称
  posterUrl: string;      // 海报URL
  bio: string;            // 个人简介
  liveUrl: string;        // 直播间URL
  streamerType: StreamerType;  // 主播类型（枚举）
  isStar: boolean;         // 是否为明星主播（明星主播可以是驴酱主播或嘉宾主播）
}
```

### 3.3 API 接口设计

需要新增以下API接口：

- `GET /api/streamers` - 获取所有主播列表
- `GET /api/streamers/:id` - 获取单个主播详情
- `POST /api/streamers` - 创建主播
- `PATCH /api/streamers/:id` - 更新主播
- `DELETE /api/streamers/:id` - 删除主播

### 5.2 服务层集成

新增 `streamerService.ts` 文件，提供主播数据获取功能：

```typescript
import axios from './axios';
import type { Streamer } from './types';

export const streamerService = {
  async getAll(): Promise<Streamer[]> {
    const response = await axios.get('/api/streamers');
    return response.data.data;
  },

  async getById(id: string): Promise<Streamer> {
    const response = await axios.get(`/api/streamers/${id}`);
    return response.data.data;
  },

  async create(data: Partial<Streamer>): Promise<Streamer> {
    const response = await axios.post('/api/streamers', data);
    return response.data.data;
  },

  async update(id: string, data: Partial<Streamer>): Promise<Streamer> {
    const response = await axios.patch(`/api/streamers/${id}`, data);
    return response.data.data;
  },

  async remove(id: string): Promise<void> {
    await axios.delete(`/api/streamers/${id}`);
  }
};
```

### 9.2 API 控制器

```typescript
@Controller('api/streamers')
export class StreamersController {
  constructor(private readonly streamersService: StreamersService) {}

  @Get()
  async findAll() {
    return this.streamersService.findAll();
  }

  @Get(':id')
  async findOne(@Param('id') id: string) {
    return this.streamersService.findOne(id);
  }

  @Post()
  async create(@Body() createStreamerDto: CreateStreamerDto) {
    return this.streamersService.create(createStreamerDto);
  }

  @Patch(':id')
  async update(@Param('id') id: string, @Body() updateStreamerDto: UpdateStreamerDto) {
    return this.streamersService.update(id, updateStreamerDto);
  }

  @Delete(':id')
  async remove(@Param('id') id: string) {
    return this.streamersService.remove(id);
  }
}
```
