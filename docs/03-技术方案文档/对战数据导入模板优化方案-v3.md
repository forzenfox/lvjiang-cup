# 对战数据导入模板优化方案

> **文档版本**：v3.0
> **创建日期**：2026-04-25
> **用途**：导入模板优化修改方案，待审核
> **修改目标**：
> 1. 添加视频BV号（大小写敏感），数据库直接存储BV号
> 2. 移除游戏时长字段（页面不展示，无意义）
> 3. 移除一血字段（无实际业务价值）

---

## 一、需求概述

### 1.1 添加视频BV号

- **需求**：在导入模板中增加视频BV号字段
- **大小写敏感**：BV号严格区分大小写（如 `BV1Ab4y1X7zK` vs `BV1aB4Y1x7Zk`）
- **存储方式**：数据库直接存储原始BV号字符串，不做转换

### 1.2 移除游戏时长

- **需求**：从导入模板、数据库和页面展示中移除游戏时长字段
- **原因**：该字段在页面中不展示，无实际业务价值

### 1.3 移除一血字段

- **需求**：从导入模板、数据库和页面展示中移除一血字段
- **原因**：该字段无实际业务价值，且增加导入复杂度

**3.3.3 数据验证修改（match-excel.util.ts）**

```typescript
// validateMatchInfo 函数修改
export function validateMatchInfo(data: MatchInfoData): ValidationResult {
  const errors: string[] =

  if (!data.redTeamName) {
    errors.push('红方战队名称不能为空');
  }
  if (!data.blueTeamName) {
    errors.push('蓝方战队名称不能为空');
  }
  if (data.gameNumber < 1 || data.gameNumber > 5) {
    errors.push('局数必须在1-5之间');
  }
  if (!data.gameStartTime) {
    errors.push('游戏开始时间不能为空');
  }
  // 移除游戏时长验证
  // if (!data.gameDuration) { ... }
  if (!data.winner) {
    errors.push('获胜方不能为空');
  } else if (!['red', 'blue', 'Red', 'Blue', '红方', '蓝方'].includes(data.winner)) {
    errors.push('获胜方必须是 red 或 blue');
  }
  // 新增BV号格式验证（如果填写）
  if (data.videoBvid && !isValidBVId(data.videoBvid)) {
    errors.push('视频BV号格式错误，应为 BVxxxxxxxxxx 格式');
  }
  return { valid: errors.length === 0, errors };
}

// 新增BV号验证函数
function isValidBVId(bvid: string): boolean {
  // BV号格式：以BV开头，后接10位大小写字母和数字
  return /^BV[a-zA-Z0-9]{10}$/.test(bvid);
}
```

## 八、注意事项

### 8.1 BV号规范

- BV号格式：`BV` + 10位大小写字母和数字
- 示例：`BV1Ab4y1X7zK`
- 大小写敏感，存储原始值
- 非必填字段

### 8.2 废弃字段处理

- 游戏时长和一血字段仅标记废弃，不删除
- 导入时游戏时长插入空字符串
- 导入时一血固定为0
- 查询时保留字段返回（兼容旧前端）

### 8.3 模板刷新

- 实施后需要刷新模板文件
- 删除旧模板：`templates/match-data-import-template.xlsx`
- 系统会自动生成新模板
