# 英雄联盟赛事官网 - 英雄头像展示完整技术方案文档
## 文档说明
本文档基于 **Riot 官方 Data Dragon** 公开数据源设计，实现**前端自动获取最新版本 → 拉取全英雄数据 → 根据中文英雄名生成头像 URL** 的完整流程。
方案稳定、官方、无鉴权、国内可直接使用，适用于赛事对局、选手英雄池、BP 界面等场景。

---

# 一、方案整体流程
1. **请求官方版本接口** → 获取最新游戏版本号（如 `16.7.1`）
2. **拼接英雄数据接口** → 获取该版本全量英雄标准数据
3. **字段筛选** → 只保留业务需要的关键字段（英文名、中文名、称号）
4. **构建映射表** → 生成 `中文名 → 英文名` 映射
5. **生成头像 URL** → 拼接官方 CDN 地址，页面直接渲染

# 二、官方公开接口（全部可直接访问）
## 1. 获取最新版本号（必用）
```
https://ddragon.leagueoflegends.com/api/versions.json
```
- 返回：版本数组，**第一个为最新版本**
- 示例：`["16.7.1", "16.6.1", ...]`

## 2. 获取全英雄数据（中文）
```
https://ddragon.leagueoflegends.com/cdn/{版本}/data/zh_CN/champion.json
```
- 示例（最新）：
```
https://ddragon.leagueoflegends.com/cdn/16.7.1/data/zh_CN/champion.json
```

## 3. 英雄头像 CDN（官方全球加速）
```
https://ddragon.leagueoflegends.com/cdn/{版本}/img/champion/{英文名}.png
```
- 示例：`https://ddragon.leagueoflegends.com/cdn/16.7.1/img/champion/Yasuo.png`

# 三、英雄数据字段说明（仅保留业务需要）
接口返回数据结构如下，**只标注你需要关心的字段**，其余可忽略不解析。

```json
{
  "version": "16.7.1",         // 当前数据版本
  "data": {                    // 英雄集合
    "Yasuo": {                 // 【Key = 英文ID → 拼头像用】
      "id": "Yasuo",           // 唯一ID（= 英文名）
      "key": "157",            // 数字ID（游戏内ID）
      "name": "亚索",          // 【中文名称 → 页面展示】
      "title": "疾风剑豪",     // 【英雄称号】
      "tags": ["Fighter"],     // 【定位：战士/法师/射手/辅助/坦克/刺客】
      "image": {               // 头像信息（可直接用 id 替代）
        "full": "Yasuo.png"
      }
    }
  }
}
```

## 你的业务模型建议（极简）
```javascript
{
  enName: "Yasuo",      // 英文ID（拼头像用）
  cnName: "亚索",       // 中文名称
```

# 四、前端完整实现代码（可直接复制使用）
## 1. 工具类：lol-champion-api.js
```javascript
/**
 * 获取 Data Dragon 最新版本号
 */
export async function getLatestVersion() {
  try {
    const res = await fetch('https://ddragon.leagueoflegends.com/api/versions.json')
    const versions = await res.json()
    return versions[0] // 第一个就是最新
  } catch (err) {
    console.error('获取版本失败', err)
    return '16.7.1' // 备用版本
  }
}

/**
 * 获取最新版本全英雄列表（中文）
 */
export async function getChampionList() {
  const version = await getLatestVersion()
  const url = `https://ddragon.leagueoflegends.com/cdn/${version}/data/zh_CN/champion.json`

  try {
    const res = await fetch(url)
    const json = await res.json()
    return json.data || {}
  } catch (err) {
    console.error('获取英雄数据失败', err)
    return {}
  }
}

/**
 * 构建：中文名 → 英文名 映射
 */
export async function buildCnToEnMap() {
  const champions = await getChampionList()
  const map = {}

  for (const enName in champions) {
    const hero = champions[enName]
    const cnName = hero.name // "亚索"
    map[cnName] = enName
  }
  return map
}

/**
 * 根据中文名称获取头像 URL
 */
export async function getChampionIcon(cnName) {
  const version = await getLatestVersion()
  const map = await buildCnToEnMap()
  const enName = map[cnName] || 'Default' // 不存在则用默认

  return `https://ddragon.leagueoflegends.com/cdn/${version}/img/champion/${enName}.png`
}
```

# 五、页面使用示例（Vue/React/原生JS通用）
```vue
<template>
  <div>
    <img :src="iconUrl" alt="英雄头像" />
    <span>{{ name }}</span>
  </div>
</template>

<script setup>
import { getChampionIcon } from './lol-champion-api.js'
import { ref, onMounted } from 'vue'

const name = ref('亚索')
const iconUrl = ref('')

onMounted(async () => {
  iconUrl.value = await getChampionIcon(name.value)
})
</script>

<style>
img { width: 80px; height: 80px; border-radius: 50%; object-fit: cover; }
</style>
```

# 六、方案优势
1. **官方数据源**：永久维护、永不失效
2. **自动最新**：每次加载自动获取最新版本与新英雄
3. **国内可访问**：CDN 加速，加载速度快
4. **极简解析**：只保留名称、定位、英文ID，不浪费流量
5. **无鉴权**：无需 API Key、无需跨域配置
6. **头像标准**：官方头像、透明底、无水印
