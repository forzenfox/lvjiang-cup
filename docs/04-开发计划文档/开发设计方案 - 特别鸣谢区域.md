## 1. 概述

### 1.1 产品目标

在主页面 ScheduleSection（赛程区域）下方新增"特别鸣谢"展示区域，用于感谢赛事赞助方和幕后工作人员。

**核心体验：**
- 赞助信息以水平弹幕方式从右向左滚动展示
- 特殊奖项以列表形式补充展示
- 幕后工作人员按角色分组静态展示
- 支持 PC 端悬停暂停、移动端持续滚动

### 1.2 技术选型

| 技术 | 方案 | 说明 |
|------|------|------|
| **弹幕动画** | CSS `@keyframes` + `translateX` | GPU 加速，性能最优 |
| **前端框架** | React 18 + TypeScript + Tailwind CSS | 与现有项目一致 |
| **后端框架** | NestJS + SQLite | 复用现有架构 |
| **UI 组件** | shadcn/ui (Button/Card/Toast) | 后台管理使用现有组件库 |
| **HTTP 客户端** | Axios | 复用现有 apiClient |
| **数据缓存** | localStorage | 前端缓存 10 分钟 |

### 1.3 关键设计决策

#### 决策 1：弹幕滚动方案

**问题：** 如何实现无缝循环的水平弹幕滚动？

**方案：** 内容复制 + CSS Animation `translateX(-50%)`

**理由：**
- ✅ 纯 CSS 实现，利用 GPU 硬件加速，性能最优
- ✅ 避免 JavaScript 定时器带来的性能问题和掉帧
- ✅ 实现简单，易于维护
- ❌ 需要动态计算动画时长

**技术实现：**
```css
@keyframes marquee-scroll {
  0%   { transform: translateX(0); }
  100% { transform: translateX(-50%); }
}
```

#### 决策 2：前端展示数据结构

**问题：** 弹幕滚动和特殊奖项列表是分开两个接口还是合并一个接口？

**方案：** 赞助信息和工作人员各一个接口，前端根据 `specialAward` 字段判断是否展示在下方列表。

**理由：**
- ✅ 接口职责清晰，前端数据处理简单
- ✅ 后端只需维护两张表
- ✅ 避免特殊奖项单独建表的复杂度

#### 决策 3：视口检测方案

**问题：** 如何实现弹幕区域进入视口时启动动画，离开视口时暂停？

**方案：** 使用 `IntersectionObserver` API

**理由：**
- ✅ 原生 API，无需额外依赖
- ✅ 性能优于 `scroll` 事件监听
- ✅ 浏览器支持度良好（Chrome 51+, Firefox 55+, Safari 12.1+）

## 3. 数据库设计

### 3.1 sponsors 表（赞助信息表）

```sql
CREATE TABLE IF NOT EXISTS sponsors (
  id TEXT PRIMARY KEY,
  sponsor_name TEXT NOT NULL,           -- 赞助人名称
  sponsor_content TEXT NOT NULL,         -- 赞助内容
  special_award TEXT,                    -- 特殊奖项说明（可选）
  "order" INTEGER DEFAULT 0,             -- 排序值
  status TEXT CHECK(status IN ('enabled', 'disabled')) DEFAULT 'enabled',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  created_by TEXT                        -- 创建人（管理员 ID）
);

-- 索引
CREATE INDEX IF NOT EXISTS idx_sponsors_order ON sponsors("order");
CREATE INDEX IF NOT EXISTS idx_sponsors_status ON sponsors(status);
```

**字段说明：**

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | TEXT | 是 | UUID | 唯一标识 |
| sponsor_name | TEXT | 是 | - | 赞助人名称（最多 30 字符） |
| sponsor_content | TEXT | 是 | - | 赞助内容（最多 50 字符） |
| special_award | TEXT | 否 | null | 特殊奖项说明（最多 100 字符） |
| order | INTEGER | 是 | 0 | 排序值，越小越靠前 |
| status | TEXT | 是 | 'enabled' | 状态：enabled/disabled |
| created_at | DATETIME | 是 | 当前时间 | 创建时间 |
| updated_at | DATETIME | 是 | 当前时间 | 更新时间 |
| created_by | TEXT | 否 | null | 创建人 |

### 4.3 Service 层

```typescript
// backend/src/modules/thanks/thanks.service.ts
import { Injectable } from '@nestjs/common';
import { DatabaseService } from '../../database/database.service';
import { v4 as uuidv4 } from 'uuid';

@Injectable()
export class ThanksService {
  constructor(private db: DatabaseService) {}

  // 前端接口
  async getSponsors() {
    const rows = await this.db.all(
      `SELECT id, sponsor_name as sponsorName, sponsor_content as sponsorContent, special_award as specialAward
       FROM sponsors
       WHERE status = 'enabled'
       ORDER BY "order" ASC, created_at ASC`,
    );
    return { list: rows };
  }

  async getStaff() {
    const rows = await this.db.all(
      `SELECT id, name, role
       FROM staff
       WHERE status = 'enabled'
       ORDER BY "order" ASC, created_at ASC`,
    );
    return { list: rows };
  }

  // 后台接口
  async getAdminSponsors(params: { page?: number; pageSize?: number; keyword?: string; status?: string }) {
    const { page = 1, pageSize = 20, keyword, status } = params;
    let whereClause = 'WHERE 1=1';
    const queryParams: any[] = [];

    if (keyword) {
      whereClause += ' AND sponsor_name LIKE ?';
      queryParams.push(`%${keyword}%`);
    }
    if (status !== undefined && status !== null) {
      whereClause += ' AND status = ?';
      queryParams.push(status);
    }

    const total = await this.db.get(
      `SELECT COUNT(*) as count FROM sponsors ${whereClause}`,
      queryParams,
    );

    const rows = await this.db.all(
      `SELECT * FROM sponsors ${whereClause} ORDER BY "order" ASC, created_at ASC LIMIT ? OFFSET ?`,
      [...queryParams, pageSize, (page - 1) * pageSize],
    );

    return {
      list: rows,
      total: total.count,
      page,
      pageSize,
    };
  }

  async createSponsor(data: { sponsorName: string; sponsorContent: string; specialAward?: string; order?: number }) {
    const id = uuidv4();
    await this.db.run(
      `INSERT INTO sponsors (id, sponsor_name, sponsor_content, special_award, "order") VALUES (?, ?, ?, ?, ?)`,
      [id, data.sponsorName, data.sponsorContent, data.specialAward || null, data.order ?? 0],
    );
    return { id };
  }

  async updateSponsor(id: string, data: Partial<{ sponsorName: string; sponsorContent: string; specialAward: string; order: number; status: string }>) {
    const fields: string[] = [];
    const values: any[] = [];

    if (data.sponsorName !== undefined) { fields.push('sponsor_name = ?'); values.push(data.sponsorName); }
    if (data.sponsorContent !== undefined) { fields.push('sponsor_content = ?'); values.push(data.sponsorContent); }
    if (data.specialAward !== undefined) { fields.push('special_award = ?'); values.push(data.specialAward); }
    if (data.order !== undefined) { fields.push('"order" = ?'); values.push(data.order); }
    if (data.status !== undefined) { fields.push('status = ?'); values.push(data.status); }

    if (fields.length === 0) throw new Error('No fields to update');

    fields.push('updated_at = CURRENT_TIMESTAMP');
    values.push(id);

    await this.db.run(`UPDATE sponsors SET ${fields.join(', ')} WHERE id = ?`, values);
    return { message: '更新成功' };
  }

  async deleteSponsor(id: string) {
    await this.db.run(`DELETE FROM sponsors WHERE id = ?`, [id]);
    return { message: '删除成功' };
  }

  async sortSponsors(ids: string[]) {
    const db = this.db.getDatabase(); // 获取原生 SQLite 实例
    await new Promise<void>((resolve, reject) => {
      db.serialize(() => {
        db.run('BEGIN TRANSACTION');
        const stmt = db.prepare(`UPDATE sponsors SET "order" = ? WHERE id = ?`);
        ids.forEach((id, index) => {
          stmt.run(index, id);
        });
        stmt.finalize();
        db.run('COMMIT', (err) => (err ? reject(err) : resolve()));
      });
    });
    return { message: '排序更新成功' };
  }

  // staff 方法同理...
}
```

```typescript
  const [error, setError] = useState<string | null>(null);

    const fetchData = async () => {
      try {
        setLoading(true);
        // 尝试从缓存读取
        const cached = localStorage.getItem('thanks_data');
        if (cached) {
          const { data, timestamp } = JSON.parse(cached);
          if (Date.now() - timestamp < 10 * 60 * 1000) {
            setSponsors(data.sponsors);
            setStaff(data.staff);
            setLoading(false);
            return;
          }
        }

        const [sponsorsRes, staffRes] = await Promise.all([getSponsors(), getStaff()]);
        setSponsors(sponsorsRes.list);
        setStaff(staffRes.list);

        // 写入缓存
        localStorage.setItem('thanks_data', JSON.stringify({
          data: { sponsors: sponsorsRes.list, staff: staffRes.list },
          timestamp: Date.now(),
        }));
      } catch (err) {
        setError(err instanceof Error ? err.message : '加载失败');
      } finally {
        setLoading(false);
      }
    };

    fetchData();
  }, []);

  if (loading) return <SkeletonLoader />;
  if (error) return <ErrorState onRetry={() => window.location.reload()} />;

  return (
    <section id="thanks" className="min-h-screen py-20 bg-gradient-to-b from-black to-background relative">
```

```typescript
// frontend/src/components/features/ThanksSection/MarqueeBanner.tsx
import React, { useRef, useState, useEffect } from 'react';
import type { SponsorItem } from '@/api/thanks';

interface MarqueeBannerProps {
  sponsors: SponsorItem[];
}

export const MarqueeBanner: React.FC<MarqueeBannerProps> = ({ sponsors }) => {
  const containerRef = useRef<HTMLDivElement>(null);
  const contentRef = useRef<HTMLDivElement>(null);
  const [isPaused, setIsPaused] = useState(false);
  const [scrollDuration, setScrollDuration] = useState(15);
  const [inViewport, setInViewport] = useState(false);

  // 视口检测
  useEffect(() => {
    const observer = new IntersectionObserver(
      ([entry]) => setInViewport(entry.isIntersecting),
      { threshold: 0.1 },
    );

    if (containerRef.current) observer.observe(containerRef.current);
    return () => observer.disconnect();
  }, []);

  // 动态计算滚动时长
  useEffect(() => {
    if (contentRef.current && containerRef.current) {
      const contentWidth = contentRef.current.scrollWidth / 2; // 复制后宽度的一半
      const viewportWidth = containerRef.current.offsetWidth;
      const baseTime = window.innerWidth >= 768 ? 15 : 12; // PC 15s, 移动 12s
      const duration = (contentWidth / viewportWidth) * baseTime;
      setScrollDuration(duration);
    }
  }, [sponsors]);

  if (sponsors.length === 0) {
    return (
      <div className="h-[60px] md:h-[60px] flex items-center justify-center border-y border-amber-500/30 bg-black/60 rounded-lg">
        <p className="text-amber-400 text-lg">感谢所有支持驴酱杯的朋友们</p>
      </div>
    );
  }

  const marqueeContent = (
    <div className="flex items-center gap-20 px-10" ref={contentRef}>
      {sponsors.map((s) => (
        <span key={s.id} className="text-amber-400 text-lg font-medium whitespace-nowrap">
          感谢老板{s.sponsorName}赞助的{s.sponsorContent}，老板大气
        </span>
      ))}
    </div>
  );

  return (
    <div
      ref={containerRef}
      className="relative h-[60px] overflow-hidden border-y border-amber-500/30 bg-black/60 rounded-lg"
      onMouseEnter={() => setIsPaused(true)}
      onMouseLeave={() => setIsPaused(false)}
    >
      <div
        className="flex items-center gap-0 will-change-transform"
        style={{
          animation: `marquee-scroll ${scrollDuration}s linear infinite`,
          animationPlayState: inViewport && !isPaused ? 'running' : 'paused',
        }}
      >
        {/* 复制一份实现无缝循环 */}
        {marqueeContent}
        {marqueeContent}
      </div>

      <style>{`
        @keyframes marquee-scroll {
          0%   { transform: translateX(0); }
          100% { transform: translateX(-50%); }
        }
      `}</style>
    </div>
  );
};
```

## 6. 弹幕滚动核心技术

### 6.1 方案选型对比

| 方案 | 实现方式 | 优点 | 缺点 | 选择 |
|------|----------|------|------|------|
| CSS Animation | `@keyframes` + `translateX` | GPU 加速、性能好、实现简单 | 动态计算时长需 JS 辅助 | ✅ 推荐 |
| JS 定时器 | `setInterval` + `translateX` | 控制精确 | 性能较差、可能掉帧 | ❌ |
| Framer Motion | `animate` + `x` | 与项目技术栈一致 | 弹幕场景过重 | ❌ |
| CSS Marquee | `<marquee>` 标签 | 最简单 | 已废弃、不推荐 | ❌ |

### 6.2 核心实现原理

```
┌──────────────────────────────────────────────────────────────┐
│                     视口 (viewport)                           │
│                                                              │
│  ┌──────────────────┬──────────────────┐                     │
│  │ 第一份内容 (A)    │ 第二份内容 (A')   │ ← 复制一份          │
│  │ 赞助人1 赞助人2   │ 赞助人1 赞助人2   │                     │
│  └──────────────────┴──────────────────┘                     │
│  ←────────────────── 内容总宽度 ──────────────────→           │
│                                                              │
│  动画: translateX(0) → translateX(-50%)                      │
│  当 A 完全移出视口时，A' 刚好到达起始位置 → 无缝衔接            │
└──────────────────────────────────────────────────────────────┘
```

### 6.3 CSS 实现

```css
@keyframes marquee-scroll {
  0% {
    transform: translateX(0);
  }
  100% {
    transform: translateX(-50%);
  }
}

.marquee-container {
  overflow: hidden;
  position: relative;
}

.marquee-content {
  display: flex;
  will-change: transform;
  animation: marquee-scroll var(--scroll-duration) linear infinite;
}

.marquee-container:hover .marquee-content {
  animation-play-state: paused;
}
```

### 6.4 动态时长计算

```typescript
const calculateScrollDuration = () => {
  if (!contentRef.current || !containerRef.current) return 15;

  const contentWidth = contentRef.current.scrollWidth / 2; // 单份宽度
  const viewportWidth = containerRef.current.offsetWidth;
```

### 6.5 性能优化要点

| 优化点 | 实现方式 | 效果 |
|--------|----------|------|
| GPU 加速 | `transform` + `will-change: transform` | 避免重排重绘 |
| 视口控制 | `IntersectionObserver` 启停动画 | 减少不必要的计算 |
| 悬停暂停 | `animation-play-state: paused` | 提升用户体验 |
| 移动端 | 不提供悬停暂停，持续滚动 | 减少触摸事件处理 |

```typescript
const SponsorForm: React.FC<SponsorFormProps> = ({ initialData, onSave, onClose }) => {
  const [formData, setFormData] = useState<SponsorFormData>({
    sponsorName: initialData?.sponsorName || '',
    sponsorContent: initialData?.sponsorContent || '',
    specialAward: initialData?.specialAward || '',
    order: initialData?.order ?? 0,
  });
  const [errors, setErrors] = useState<Partial<Record<keyof SponsorFormData, string>>>({});

  const validate = (): boolean => {
    const newErrors: typeof errors = {};
    if (!formData.sponsorName.trim()) newErrors.sponsorName = '赞助人名称不能为空';
    if (formData.sponsorName.length > 30) newErrors.sponsorName = '赞助人名称最多 30 字符';
    if (!formData.sponsorContent.trim()) newErrors.sponsorContent = '赞助内容不能为空';
    if (formData.sponsorContent.length > 50) newErrors.sponsorContent = '赞助内容最多 50 字符';
    if (formData.specialAward && formData.specialAward.length > 100) newErrors.specialAward = '特殊奖项说明最多 100 字符';
    setErrors(newErrors);
    return Object.keys(newErrors).length === 0;
  };

  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault();
    if (!validate()) return;
    await onSave(formData);
  };

  return (
    <div className="fixed inset-0 bg-black/50 flex items-center justify-center z-50">
      <div className="bg-gray-900 rounded-lg p-6 w-full max-w-md">
        <div className="flex items-center justify-between mb-4">
          <h2 className="text-xl font-bold text-white">
            {initialData ? '编辑赞助信息' : '添加赞助信息'}
          </h2>
          <Button variant="ghost" size="sm" onClick={onClose}>
            <X className="w-4 h-4" />
          </Button>
        </div>

        <form onSubmit={handleSubmit} className="space-y-4">
          <div>
            <label className="text-sm text-gray-300">赞助人名称 *</label>
            <Input
              value={formData.sponsorName}
              onChange={(e) => setFormData({ ...formData, sponsorName: e.target.value })}
              maxLength={30}
              placeholder="例如：秀木老板"
            />
            {errors.sponsorName && <p className="text-red-400 text-xs mt-1">{errors.sponsorName}</p>}
          </div>

          <div>
            <label className="text-sm text-gray-300">赞助内容 *</label>
            <Input
              value={formData.sponsorContent}
              onChange={(e) => setFormData({ ...formData, sponsorContent: e.target.value })}
              maxLength={50}
              placeholder="例如：2W"
            />
            {errors.sponsorContent && <p className="text-red-400 text-xs mt-1">{errors.sponsorContent}</p>}
          </div>

          <div>
            <label className="text-sm text-gray-300">特殊奖项说明（可选）</label>
            <Textarea
              value={formData.specialAward || ''}
              onChange={(e) => setFormData({ ...formData, specialAward: e.target.value })}
              maxLength={100}
              placeholder="例如：4强每人一份贡菜千层肚"
            />
            {errors.specialAward && <p className="text-red-400 text-xs mt-1">{errors.specialAward}</p>}
          </div>

          <div className="flex justify-end gap-2">
            <Button variant="outline" onClick={onClose}>取消</Button>
            <Button type="submit">保存</Button>
          </div>
        </form>
      </div>
    </div>
  );
};

export default SponsorForm;
```

## 9. 性能优化

### 9.1 弹幕滚动优化

| 优化项 | 实现方式 | 效果 |
|--------|----------|------|
| GPU 加速 | `transform` + `will-change: transform` | 避免重排重绘，帧率稳定 60fps |
| 视口检测 | `IntersectionObserver` | 区域离屏时暂停动画，节省 CPU |
| 防抖 | 窗口 resize 事件重新计算滚动时长 | 避免频繁重算 |
| 内容复用 | React.memo 包裹 MarqueeBanner | 避免 sponsors 未变化时重新渲染 |

### 9.2 数据缓存

```typescript
// 10 分钟缓存
const CACHE_TTL = 10 * 60 * 1000;

const getCachedData = () => {
  const cached = localStorage.getItem('thanks_data');
  if (cached) {
    const { data, timestamp } = JSON.parse(cached);
    if (Date.now() - timestamp < CACHE_TTL) return data;
  }
  return null;
};
```

### 9.3 图片/资源优化

本项目不涉及图片资源，纯文字弹幕，无需额外优化。
