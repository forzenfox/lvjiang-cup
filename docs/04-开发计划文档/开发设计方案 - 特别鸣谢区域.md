## 1. 概述

### 1.1 产品目标

在主页面 ScheduleSection（赛程区域）下方新增"特别鸣谢"展示区域，用于感谢赛事赞助方和幕后工作人员。

**核心体验：**
- 赞助信息以水平弹幕方式从右向左滚动展示
- 特殊奖项以列表形式补充展示
- 幕后工作人员按角色分组静态展示
- 支持 PC 端悬停暂停、移动端持续滚动

### 1.2 数据方案决策

**决策：采用纯前端 JSON 配置文件，不依赖后端 API 和数据库。**

**理由：**
- ✅ 驴酱杯赞助名单已固定，赛事结束后基本不会变动
- ✅ 更新频率极低，修改配置文件即可
- ✅ 开发成本从 16-24 小时降低到 4-6 小时
- ✅ 无需后端表设计、API 开发、后台管理页面
- ❌ 每次更新需要修改代码并提交（但更新频率低，可接受）

### 1.3 技术选型

| 技术 | 方案 | 说明 |
|------|------|------|
| **弹幕动画** | CSS `@keyframes` + `translateX` | GPU 加速，性能最优 |
| **前端框架** | React 18 + TypeScript + Tailwind CSS | 与现有项目一致 |
| **数据源** | JSON 配置文件 | 纯前端静态数据，无需后端 |
| **视口检测** | `IntersectionObserver` | 原生 API，无需额外依赖 |
| **UI 组件** | 自定义组件 + Tailwind | 无需引入额外 UI 库 |

### 4.2 MarqueeBanner 弹幕组件

```typescript
// frontend/src/components/features/ThanksSection/MarqueeBanner.tsx
import React, { useRef, useState, useEffect } from 'react';
import type { SponsorConfig } from '@/data/types';

interface MarqueeBannerProps {
  sponsors: SponsorConfig[];
}

export const MarqueeBanner: React.FC<MarqueeBannerProps> = ({ sponsors }) => {
  const containerRef = useRef<HTMLDivElement>(null);
  const contentRef = useRef<HTMLDivElement>(null);
  const [isPaused, setIsPaused] = useState(false);
  const [scrollDuration, setScrollDuration] = useState(15);
  const [inViewport, setInViewport] = useState(false);

  // 视口检测
  useEffect(() => {
    const observer = new IntersectionObserver(
      ([entry]) => setInViewport(entry.isIntersecting),
      { threshold: 0.1 },
    );

    if (containerRef.current) observer.observe(containerRef.current);
    return () => observer.disconnect();
  }, []);

  // 动态计算滚动时长
  useEffect(() => {
    const updateDuration = () => {
      if (contentRef.current && containerRef.current) {
        const contentWidth = contentRef.current.scrollWidth / 2;
        const viewportWidth = containerRef.current.offsetWidth;
        const baseTime = window.innerWidth >= 768 ? 15 : 12;
        const duration = (contentWidth / viewportWidth) * baseTime;
        setScrollDuration(Math.max(duration, 5));
      }
    };

    updateDuration();
    window.addEventListener('resize', updateDuration);
    return () => window.removeEventListener('resize', updateDuration);
  }, [sponsors]);

  if (sponsors.length === 0) {
    return (
      <div className="h-[44px] md:h-[60px] flex items-center justify-center border-y border-amber-500/30 bg-black/60 rounded-lg">
        <p className="text-amber-400 text-[14px] md:text-lg">感谢所有支持驴酱杯的朋友们</p>
      </div>
    );
  }

  const marqueeContent = (
    <div className="flex items-center gap-16 md:gap-20 px-8 md:px-10" ref={contentRef}>
      {sponsors.map((s) => (
        <span key={s.id} className="text-amber-400 text-[14px] md:text-lg font-medium whitespace-nowrap">
          感谢老板{s.sponsorName}赞助的{s.sponsorContent}，老板大气
        </span>
      ))}
    </div>
  );

  return (
    <div
      ref={containerRef}
      className="relative h-[44px] md:h-[60px] overflow-hidden border-y border-amber-500/30 bg-black/60 rounded-lg"
      onMouseEnter={() => setIsPaused(true)}
      onMouseLeave={() => setIsPaused(false)}
    >
      <div
        className="flex items-center gap-0 will-change-transform"
        style={{
          animation: `marquee-scroll ${scrollDuration}s linear infinite`,
          animationPlayState: inViewport && !isPaused ? 'running' : 'paused',
        }}
      >
        {marqueeContent}
        {marqueeContent}
      </div>

      <style>{`
        @keyframes marquee-scroll {
          0%   { transform: translateX(0); }
          100% { transform: translateX(-50%); }
        }
      `}</style>
    </div>
  );
};
```

### 4.3 SpecialAwards 特殊奖项组件

```typescript
// frontend/src/components/features/ThanksSection/SpecialAwards.tsx
import React, { useState } from 'react';
import type { SponsorConfig } from '@/data/types';

interface SpecialAwardsProps {
  sponsors: SponsorConfig[];
}

export const SpecialAwards: React.FC<SpecialAwardsProps> = ({ sponsors }) => {
  const [isExpanded, setIsExpanded] = useState(false);

  const awards = sponsors.filter((s) => s.specialAward);
  if (awards.length === 0) return null;

  const isMobile = typeof window !== 'undefined' && window.innerWidth < 768;
  const displayAwards = isMobile && !isExpanded ? awards.slice(0, 3) : awards;

  return (
    <div className="mt-6 md:mt-8 p-4 md:p-6 bg-black/40 rounded-lg border border-white/10">
      <h3 className="text-base md:text-lg font-semibold text-amber-400 mb-3 md:mb-4">特殊奖项</h3>
      <ul className="space-y-2">
        {displayAwards.map((a) => (
          <li key={a.id} className="flex items-start gap-2 text-gray-300 text-xs md:text-sm">
            <span className="text-amber-400 mt-0.5">•</span>
            <span>
              <strong className="text-white">{a.sponsorName}</strong>：{a.specialAward}
            </span>
          </li>
        ))}
      </ul>

      {isMobile && awards.length > 3 && (
        <button
          className="mt-3 md:mt-4 text-amber-400 hover:text-amber-300 text-xs md:text-sm"
          onClick={() => setIsExpanded(!isExpanded)}
        >
          {isExpanded ? '收起' : '查看更多'} ({awards.length - 3} 条)
        </button>
      )}
    </div>
  );
};
```

### 4.4 StaffThanks 工作人员组件

```typescript
// frontend/src/components/features/ThanksSection/StaffThanks.tsx
import React from 'react';
import type { StaffConfig } from '@/data/types';

interface StaffThanksProps {
  staff: StaffConfig[];
}

export const StaffThanks: React.FC<StaffThanksProps> = ({ staff }) => {
  if (staff.length === 0) return null;

  // 按角色分组
  const grouped = staff.reduce<Record<string, string[]>>((acc, s) => {
    if (!acc[s.role]) acc[s.role] = [];
    acc[s.role].push(s.name);
    return acc;
  }, {});

  return (
    <div className="mt-6 md:mt-8 p-4 md:p-6 bg-black/40 rounded-lg border border-white/10">
      <h3 className="text-base md:text-lg font-semibold text-amber-400 mb-3 md:mb-4">幕后工作人员</h3>
      <div className="space-y-2 md:space-y-3">
        {Object.entries(grouped).map(([role, names]) => (
          <p key={role} className="text-gray-300 text-xs md:text-sm md:text-base">
            感谢<span className="text-amber-400 font-medium">{role}</span>：{names.join('、')}的辛勤付出
          </p>
        ))}
      </div>
    </div>
  );
};
```

## 5. 弹幕滚动核心技术

### 5.1 方案选型对比

| 方案 | 实现方式 | 优点 | 缺点 | 选择 |
|------|----------|------|------|------|
| CSS Animation | `@keyframes` + `translateX` | GPU 加速、性能好、实现简单 | 动态计算时长需 JS 辅助 | ✅ 推荐 |
| JS 定时器 | `setInterval` + `translateX` | 控制精确 | 性能较差、可能掉帧 | ❌ |
| Framer Motion | `animate` + `x` | 与项目技术栈一致 | 弹幕场景过重 | ❌ |
| CSS Marquee | `<marquee>` 标签 | 最简单 | 已废弃、不推荐 | ❌ |

### 5.2 核心实现原理

```
┌──────────────────────────────────────────────────────────────┐
│                     视口 (viewport)                           │
│                                                              │
│  ┌──────────────────┬──────────────────┐                     │
│  │ 第一份内容 (A)    │ 第二份内容 (A')   │ ← 复制一份          │
│  │ 赞助人1 赞助人2   │ 赞助人1 赞助人2   │                     │
│  └──────────────────┴──────────────────┘                     │
│  ←────────────────── 内容总宽度 ──────────────────→           │
│                                                              │
│  动画: translateX(0) → translateX(-50%)                      │
│  当 A 完全移出视口时，A' 刚好到达起始位置 → 无缝衔接            │
└──────────────────────────────────────────────────────────────┘
```

### 5.3 CSS 实现

```css
@keyframes marquee-scroll {
  0% {
    transform: translateX(0);
  }
  100% {
    transform: translateX(-50%);
  }
}

.marquee-container {
  overflow: hidden;
  position: relative;
}

.marquee-content {
  display: flex;
  will-change: transform;
  animation: marquee-scroll var(--scroll-duration) linear infinite;
}

.marquee-container:hover .marquee-content {
  animation-play-state: paused;
}
```

### 5.4 动态时长计算

```typescript
const calculateScrollDuration = () => {
  if (!contentRef.current || !containerRef.current) return 15;

  const contentWidth = contentRef.current.scrollWidth / 2; // 单份宽度
  const viewportWidth = containerRef.current.offsetWidth;
  const baseTime = window.innerWidth >= 768 ? 15 : 12; // 单屏滚动时间

  return Math.max((contentWidth / viewportWidth) * baseTime, 5); // 最少 5 秒
};
```

### 5.5 性能优化要点

| 优化点 | 实现方式 | 效果 |
|--------|----------|------|
| GPU 加速 | `transform` + `will-change: transform` | 避免重排重绘 |
| 视口控制 | `IntersectionObserver` 启停动画 | 减少不必要的计算 |
| 悬停暂停 | `animation-play-state: paused` | 提升用户体验 |
| 移动端 | 不提供悬停暂停，持续滚动 | 减少触摸事件处理 |
| Resize 防抖 | window resize 重新计算时长 | 适配屏幕变化 |

## 6. 响应式设计

### 6.1 适配断点

| 断点 | 屏幕宽度 | 布局方案 | 说明 |
|------|----------|----------|------|
| PC 端 | ≥ 768px | 标准布局 | 弹幕 60px 高，悬停暂停 |
| 移动端 | < 768px | 紧凑布局 | 弹幕 44px 高，持续滚动 |

### 6.2 关键差异

| 元素 | PC 端 | 移动端 |
|------|-------|--------|
| 区域最小高度 | 400px (min-h-[400px]) | 400px |
| 上下内边距 | py-20 | py-12 |
| 区域标题大小 | text-3xl (28px) | text-[22px] |
| 标题间距 | mb-12 | mb-8 |
| 弹幕容器高度 | 60px | 44px |
| 弹幕文字大小 | text-lg (18px) | text-[14px] |
| 弹幕内边距 | px-10 | px-8 |
| 弹幕条目间距 | gap-20 | gap-16 |
| 弹幕滚动速度 | 单屏 15 秒 | 单屏 12 秒 |
| 悬停暂停 | 支持 | 不支持 |
| 特殊奖项内边距 | p-6 | p-4 |
| 特殊奖项文字 | text-sm (14px) | text-xs (12px) |
| 特殊奖项列表 | 全部展示 | 默认收起（前3条） |
| 工作人员文字 | text-sm ~ text-base | text-xs ~ text-sm |

### 6.3 Tailwind 响应式类示例

```tsx

// 弹幕文字大小
className="text-[14px] md:text-lg"

// 区域标题
```

## 7. 性能优化

### 7.1 弹幕滚动优化

| 优化项 | 实现方式 | 效果 |
|--------|----------|------|
| GPU 加速 | `transform` + `will-change: transform` | 避免重排重绘，帧率稳定 60fps |
| 视口检测 | `IntersectionObserver` | 区域离屏时暂停动画，节省 CPU |
| Resize 监听 | window resize 重新计算滚动时长 | 适配屏幕变化 |
| 静态数据 | JSON 配置直接 import | 零网络请求，瞬间加载 |

### 7.2 数据加载优化

- **零网络请求**：JSON 配置直接通过 `import` 引入，编译时打包进 bundle
- **无缓存逻辑**：不需要 localStorage 缓存，数据始终最新
- **Tree Shaking**：仅导入需要的类型定义

### 7.3 渲染优化

```typescript
// React.memo 避免不必要的重渲染
export const MarqueeBanner = React.memo(({ sponsors }: MarqueeBannerProps) => {
  // ...
});
```
