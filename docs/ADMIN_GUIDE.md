## 登录管理后台

### 访问地址

在浏览器中访问：`http://<您的域名>/admin`

### 登录凭证

- **密码**：`admin123`

### 登录流程

1. 打开管理后台登录页面
2. 输入用户名和密码
3. 点击"登录"按钮
4. 登录成功后自动跳转到仪表盘

## 战队管理

### 功能说明

战队管理页面允许您添加、编辑和删除参赛战队信息。每个战队包含：
- 战队名称
- 战队队标
- 战队简介
- 5名队员（top、jungle、mid、bot、support）

### 添加新战队

1. 点击页面右上角的"**添加战队**"按钮
2. 在弹出的编辑表单中填写战队信息：
   - **战队名称**：输入战队名称（如"RNG"、"EDG"等）
   - **队标链接**：输入战队 Logo 的图片 URL
   - **战队简介**：输入战队介绍文字
3. 填写队员信息：
   - 每个位置（top、jungle、mid、bot、support）都需要填写
   - **姓名**：队员游戏 ID 或真实姓名
   - **头像链接**：队员头像图片 URL（可选）
4. 点击"**保存战队**"按钮

### 编辑战队

1. 在战队列表中找到需要编辑的战队
2. 点击该战队卡片右上角的"**编辑**"图标（铅笔形状）
3. 修改需要更新的信息
4. 点击"**保存战队**"按钮

### 删除战队

1. 在战队列表中找到需要删除的战队
2. 点击该战队卡片右上角的"**删除**"图标（垃圾桶形状）
3. 在确认对话框中点击"确定"

> ⚠️ **注意**：删除战队后，与该战队相关的比赛数据可能需要手动更新。

## 赛程管理

### 功能说明

赛程管理分为两个阶段：
- **瑞士轮（Swiss Stage）**：小组赛阶段
- **淘汰赛（Elimination Stage）**：双败淘汰制阶段

### 瑞士轮管理

#### 添加比赛

1. 切换到"**瑞士轮**"标签页
2. 点击"**添加比赛**"按钮
3. 填写比赛信息：
   - **轮次**：选择比赛轮次（如"第一轮"、"第二轮"等）
   - **战队 A**：选择第一支参赛战队
   - **战队 B**：选择第二支参赛战队
   - **战绩**：选择当前战绩（如"0-0"、"1-0"等）
   - **比赛日期**：选择比赛日期
   - **开始时间**：输入比赛开始时间
4. 点击"**保存**"按钮

#### 更新比赛结果

1. 在瑞士轮比赛列表中找到需要更新的比赛
2. 点击"**编辑**"按钮
3. 更新比赛信息：
   - **比分**：输入两队比分
   - **状态**：选择"进行中"或"已结束"
   - **胜者**：如果比赛结束，选择获胜战队
4. 点击"**保存**"按钮

### 淘汰赛管理

#### 淘汰赛赛制说明

本系统采用**双败淘汰制**：
- **胜者组**：未输过比赛的队伍
- **败者组**：输过一场比赛的队伍
- **总决赛**：胜者组冠军 vs 败者组冠军

#### 添加淘汰赛

1. 切换到"**淘汰赛**"标签页
2. 点击"**添加比赛**"按钮
3. 填写比赛信息：
   - **分组**：选择"胜者组"、"败者组"或"总决赛"
   - **轮次**：输入轮次名称（如"胜者组第一轮"）
   - **战队 A / 战队 B**：选择参赛战队
4. 点击"**保存**"按钮

### 比赛状态说明

| 状态 | 说明 | 可执行操作 |
|------|------|------------|
| 未开始 | 比赛尚未开始 | 编辑信息、设置对阵 |
| 进行中 | 比赛正在进行 | 更新比分 |
| 已结束 | 比赛已结束 | 查看结果、触发晋级 |

## 直播配置

### 功能说明

配置赛事直播信息，前台首页会显示当前直播状态和入口。

### 配置直播

1. 进入"**直播配置**"页面
2. 填写直播信息：
   - **直播标题**：输入直播标题（如"驴酱杯总决赛 - RNG vs EDG"）
   - **直播链接**：输入直播间 URL
   - **直播平台**：选择平台（斗鱼、虎牙、哔哩哔哩）
   - **正在直播**：勾选表示当前正在直播
3. 点击"**保存更改**"按钮

### 平台支持

目前支持以下直播平台：
- **斗鱼直播**
- **虎牙直播**
- **哔哩哔哩**

### 直播状态说明

- **正在直播**勾选后，前台首页会显示"观看直播"入口
- 取消勾选后，直播入口会隐藏或显示为"直播未开始"

## 常见问题

### Q: 如何修改管理员密码？

目前版本使用简单的登录验证，修改密码需要修改源代码中的验证逻辑。后续版本将添加密码修改功能。

### Q: 战队 Logo 和队员头像支持哪些格式？

支持常见的图片格式：
- JPG/JPEG
- PNG
- GIF
- WebP

建议使用 HTTPS 链接，避免混合内容警告。

### Q: 瑞士轮的战绩如何计算？

瑞士轮战绩格式为"X-Y"，其中：
- X = 胜场数
- Y = 败场数

例如：
- "1-0"表示 1 胜 0 负
- "2-1"表示 2 胜 1 负

当队伍达到 3 胜时晋级，3 负时淘汰。

### Q: 可以导入批量数据吗？

当前版本暂不支持批量导入，需要通过界面逐个添加。后续版本将考虑添加 Excel/CSV 导入功能。

### Q: 如何备份数据？

当前版本使用浏览器本地存储（LocalStorage）保存数据。建议：
1. 定期导出重要数据
2. 生产环境部署时接入后端数据库
